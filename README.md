# Hello, and welcome!

This is the Sublime Plants documentation repository. Its purpose is to be an ever-growing knowledgebase of design decisions, terms & definitions, meeting notes, and plantcare knowledge.

Information about Sublime Plants itself and what we're doing is located within the hosted documentation itself, which can be accessed at [docs.sublimeplants.com](https://docs.sublimeplants.com/).

We hope you'll join us on our mission to **demistify and democratize plantcare systems and knowledge.** This repository is core to that mission.
